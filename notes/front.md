## NPM 镜像源查看和切换

```bash
# 全局切换镜像源
npm config set registry https://registry.npmmirror.com


# 查看镜像源使用状态
npm get registry

# 全局切换官方镜像源
npm config set registry https://www.npmjs.org
```
