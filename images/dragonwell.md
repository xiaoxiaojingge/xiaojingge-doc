## 说明

- 阿里巴巴的 Dragonwell（龙井）是一款基于 OpenJDK 的、针对 Java 开发的自主研发的 JDK（Java Development Kit）。

- Dragonwell JDK 旨在提供高性能、高稳定性和优化的 Java 运行环境，并针对阿里巴巴的特定需求进行了定制和优化。
